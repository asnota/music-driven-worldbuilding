💡 Usage Tips

  • Top-K Sampling: Smaller values produce more stable results, larger values increase diversity
  • Temperature: Controls the randomness of the generation; lower values produce more conservative results
  • Batch Size: The number of images generated at once; increasing it improves diversity but consumes more resources
  • Super-Resolution: Increases the resolution of the output image but adds to the processing time
  • Prompting Tips: Use specific environmental descriptions, including lighting, weather, time of day, etc.
